Step Pager and options.

Requirements:

  • jQuery v1.4.2 or later: http://docs.jquery.com/Downloading_jQuery#Download_jQuery

Usage:

Initiate the plugin with the call below and use the markup in this demo. 

$(function(){
	$('#targetElement').stepPager({});
});

Options:

					
{
    activeItemClass: 'sp-active-item',
    activeColumnClass: 'sp-active-column',
    rowNumber: 3,
    speed: 500
}
