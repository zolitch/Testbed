A demonstration of what is possible with the use of css3 Media Queries and the Flexible Box Model.

This demo uses the Flexie.js plugin which allows the flexible box model to be used cross browser.

There appear to be a number of issues with the plugin.

  • Internet explorer errors when media queries are used in conjunction with it. This is the case with or without respond.js
  • Internet Explorer crashes when the browser window is resized. This happens with the online demos as well as my experiments.
  • There is an issue with IE whereby the plugin will only work on a server and not locally.
